# Password Security Review

## What Makes a Strong Password?

A strong password is long, unique, and contains a mix of uppercase letters, lowercase letters, numbers, and special characters.

## Why Should Passwords Be Unique?

Unique passwords prevent attackers from accessing multiple accounts if one password is leaked.

## What is MFA?

Multi-Factor Authentication is an additional security layer requiring another verification method besides a password.

## Risks of Password Reuse

- Multiple accounts can be compromised.
- Identity theft.
- Financial loss.
- Data breaches.

## Password Policy

1. Use at least 12 characters.
2. Include uppercase and lowercase letters.
3. Include numbers and symbols.
4. Avoid personal information.
5. Enable MFA whenever possible.
6. Change passwords after a suspected breach.
7. Use a password manager.
