# Cyber Threat Research

## 1. Phishing

Definition:
A phishing attack tricks users into revealing sensitive information.

Example:
Fake banking emails asking users to log in.

Prevention:
- Verify sender
- Avoid suspicious links
- Enable MFA

## 2. Malware

Definition:
Malicious software designed to damage systems.

Example:
Trojan Horse malware.

Prevention:
- Install antivirus
- Update software
- Avoid unknown downloads

## 3. Ransomware

Definition:
Malware that encrypts files and demands payment.

Example:
WannaCry attack.

Prevention:
- Regular backups
- Security updates
- Antivirus protection

## 4. Social Engineering

Definition:
Manipulating people into sharing confidential information.

Example:
Fake IT support calls.

Prevention:
- Verify identities
- Never share passwords
- Security awareness training

## 5. Data Breach

Definition:
Unauthorized access to sensitive data.

Example:
Facebook data breach.

Prevention:
- Strong passwords
- MFA
- Data encryption
